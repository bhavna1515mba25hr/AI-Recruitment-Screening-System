# 🤖 AI-Based Recruitment Screening System

An intelligent recruitment automation system that leverages Artificial Intelligence and Natural Language Processing (NLP) to screen, rank, and shortlist candidates based on their resumes and job requirements.

---

## 📌 Project Overview

Hiring the right candidate is a time-consuming process. This project simplifies recruitment by automating resume screening using machine learning techniques. It evaluates candidates based on multiple factors like skills, experience, education, and certifications, ensuring a fair and efficient hiring process.

---

## 🚀 Features

* 📄 Resume parsing and analysis
* 🧠 AI-based candidate ranking
* 🔍 Job description matching using NLP
* 📊 Weighted scoring system
* ⚡ Fast and automated screening process
* 🎯 Improved hiring accuracy

---

* Cosine Similarity
* Natural Language Processing (NLP)

---

## ⚙️ How It Works

1. Upload candidate resumes and job description
2. Preprocess text data (cleaning, tokenization)
3. Convert text into numerical format using TF-IDF
4. Calculate similarity score using cosine similarity
5. Apply weighted scoring (skills, experience, etc.)
6. Rank candidates based on final score
7. Output top shortlisted candidates
